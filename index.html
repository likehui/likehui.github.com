使用 Github 和 Hexo 快速搭建个人博客

2017-03-09

个人兴趣爱好非常广泛，喜欢捣鼓各种小东西自娱自乐。虽然都没能深入研究，但是自己的“孩子”还是很想拿出来遛遛得人一句夸奖的。所以刚学 Markdown 的时候很是有想过要搭个个人博客来玩玩，一来激励自己练习 Markdown，二来也是展示一下自己的“劳动成果”。可惜第一次尝试 Github + Jeckyll 方式没能一次成功，忙起来了也就把这事儿放一边了。最近因为微信普通公众号不支持页面内插入多个链接（想做个集合贴链接到自己的不同作品），就又想着还是自己搭个网站吧~改变策略使用 Github + Hexo 倒是很快成功了，记录一下过程吧，如果能对其他想要搭建个人博客网站的小伙伴有帮助那就更好了。
